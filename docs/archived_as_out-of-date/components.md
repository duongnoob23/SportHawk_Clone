# Components for SportHawk

## ShWelcomeVideo

To play the welcome video in a style that fits the device with the opacity overlay

## ShLogoAndTitle

To display the combined logo and "SportHawk" title required by the Welcome screen, possibly as a single PNG image or a combination of logo SVG/PNG and styled text of the application title

## ShButton

Implements the touchable button-like that trigger actions

## ShText

Display some text given parameters that name the size and name the style. Use the names to get the actual style from the globals.

## ShIconTitleDescription

Uses: ShIcon, ShText of type Title and of type Muted, layout parameter

## ShIcon

Display named icon or logo with positional and look parameters

## ShFormFieldEmail

A labelled form field for email input with email validation

## ShFormFieldPassword

A labelled form field for password (hidden) input with validation

## ShTextWithLink

Array: ShText for muted text at position, link that invokes callback

## ShFormFieldName

Labelled form field for name input and validation

## ShFormFieldDate

Labelled form field for date input, min & max, and validation

## ShFormFieldChoice

Labelled field with buttons for choice

## ShSpacer

Vertical spacing as per named space from globals

## ShFormFieldPhoto

Labelled field, icon or current photo, access device camera,

## ShBulletedList

Show a titled array of text as lines of bullet icon and text

## ShIconTextLink

ShIcon and ShText for touchable link that invokes callback

## ShBottomNav

Bottom navigation bar with 5 tab items (Home, Teams, Explore, Alerts, Profile) with icons and active state indicator

## ShTopNavHome

Top navigation for home screen with tabs (Recent, For You) and logo

## ShPostCard

Complete post card showing profile pic, username, subtitle, timestamp, post content, images, and interaction buttons (like, comment)

## ShPostHeader

Post header with profile picture, username, subtitle, timestamp and menu button

## ShPostContent

Post body text with hashtag support and formatting

## ShPostImage

Image display component for post images with rounded corners and border

## ShPostActions

Action bar with like and comment buttons showing counts

## ShFloatingActionButton

Circular floating action button with icon for creating new posts

## ShFormFieldTextArea

Multi-line text input field for longer content entry

## ShDropdownField

Dropdown selector field with label and chevron indicator

## ShDateTimePicker

Date and time selector field with calendar icon

## ShPhotoUploadArea

Upload area with image icon and text prompt for adding photos

## ShCommentCard

Comment display with profile pic, username, role, comment text and timestamp

## ShCommentInput

Text input field specifically for adding comments

## ShModalSheet

Bottom sheet modal container with title and close button

## ShModalMenuItem

Menu item for modal sheets with icon and text

## ShHeaderBar

Navigation header with back button, title, and action button

## ShDeleteButton

Styled delete button with warning color
